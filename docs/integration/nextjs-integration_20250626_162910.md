# Next.js Integration

This file documents Next.js integration for the satox-sdk.

## Installation

```bash
npm install @satox/satox-sdk
```

## Usage

```javascript
import { SatoxSDK } from '@satox/satox-sdk';

const sdk = new SatoxSDK({
  // configuration options
});
```
